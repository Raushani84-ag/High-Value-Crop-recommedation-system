An AI-powered web application that recommends the **top 3 most suitable crops** based on district-level climate and soil conditions using remote sensing.

This project combines **Agricultural Engineering knowledge + Machine Learning + Data Science** to support better crop planning and decision-making.

## 🎯 Problem Statement

Farmers often rely on traditional knowledge for crop selection, which may not always align with changing **climate conditions and soil characteristics**.

This system provides a **data-driven approach** to recommend crops using:
- Historical environmental data  
- District-level analysis  
- Machine Learning predictions  

---

## 🧠 Solution Approach

1. User selects:
   - District  
   - Season (Kharif / Rabi)

2. System retrieves:
   - Rainfall  
   - Temperature  
   - Humidity  
   - Soil pH  

3. Machine Learning model:
   - Predicts suitability probabilities for crops  

4. Output:
   - Top 3 recommended crops  
   - Suitability score (%)  
   - Explanation of recommendations
